Campaign Targeting Controls

Tree Selector

The jQuery checkboxTree plugin should be able to provide the functionality we need for the Tree Selector described below. The "check if full" and "collapse images" properties should be useful for implementing Geography targeting.

Closed State

  • Geography
  • Company
  • Job Title
  • Group
  • Demographics

Open State

  • Geography
  • Company
  • Job Title
  • Group
  • Demographics

Behavior

Initial view - all boxes unselected. Nodes with children have plus icons prepended.

When plus icon is clicked it changes to a minus icon and child nodes are shown. If any of them have children, plus icons are shown next to them as well.

When a node with children is selected, hide the plus icon and any child nodes and show the hint that node should be unselected to specify children.

If a node's child is selected, prevent that node from collapsing to hide the selection.

Adapt the hint text depending on the level of the selected node as follows.

Node LevelHint Text
ContinentAll countries selected, uncheck to specify
CountryAll states selected, uncheck to specify
StateAll regions selected, uncheck to specify

Tag Selector

Behavior

  • Geography
  • Company
  • Job Title
  • Group
  • Demographics

Use the Typeahead Bizstrap javascript plugin to show groups, companies, and job titles matching user entered text via the API. When the user has selected an entry, pop it into the textarea as a targeting criterion using Chosen's Multiple Select pattern.

Dynamically adjust the height of the textarea to accomodate the number of tags populated.
