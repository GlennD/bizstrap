Reading Optimized

This layout is designed for text heavy pages. You simply need to apply the reading-optimized class to your layout where you your text heavy elements are. This class will act as a fluid layout up until a maximum width of 600px is reached at which point it becomes fixed.

You can still use fluid-grids inside of a reading optimized layout if you so wish - see below

You can also graft a reading optimized style layout into an existing fluid grid layout by adding the reading-optimized class instead of a typical span class, eg. class="span10" you would use class="span reading-optimized"
