Animations make that page sizzle

To animate any html node simply apply the animated class to it. Then apply an effect class to it, ex: fade-in. The animation will start as soon as the effect class is applied, allowing you to trigger animations on page load, on click, or any other event you can think up

Available Effects

Possible animation effects are:
  • fade-in
  • fade-in-down
  • fade-in-left
  • fade-out
  • fade-out-down
  • hinge

Delaying Animations

You can also delay animations for extra effect. To do so simply add a delayed-x class in addtion to the other classes, where x is 1-4

The delay classes correspond to the following times:

  • delayed-025s: delay 0.25s
  • delayed-05s: delay 0.5s
  • delayed-1s: delay 1s
  • delayed-2s: delay 2s
