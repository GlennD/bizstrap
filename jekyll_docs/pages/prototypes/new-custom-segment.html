New Custom Segment

Pixel Piggybacking (optional)

Pixel piggybacking allows other systems such as ad servers to track visitors to this custom segment. Enter up to 10 pixels you wish to piggyback. Enter the pixels as URLs and do not include HTML or JavaScript (e.g. http://ad.doubleclick.net/activity;src=2524098).

Note: Image pixels only support piggybacking a single pixel. Only the first pixel will be piggybacked on pages tagged with Image Pixels. Your highest priority pixel should be entered into the Pixel 1 position. Pages instrumented with the Analyze JavaScript Tag will piggyback all pixels defined in this interface.
