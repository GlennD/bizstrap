Standard Form in a well

In a standard form, use the form-horizontal form class to save vertical space. Keep field labels right-aligned as per bootstrap default.

(optional)

For optional fields, the help-muted paragraph class and small tag can be used to add an (optional) hint below the field.

Field hint

A short field hint can be displayed in place of a field-level help button as defined in Contextual Help.
A field should never have both hint text and an in-line help button.

How to use the small form

Modals and widgets can include forms, and because of their limited horizontal space, should use a different form layout. Use the form-vertical form class that top-aligns field labels.

Field hints should go below fields. Any validation messages should be displayed beneath the field and any field hints. Do not use the in-line help button pattern within a modal or widget.
